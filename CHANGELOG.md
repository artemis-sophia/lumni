# Changelog

All notable changes to Lumni will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-01-07

### Added
- Complete Python/FastAPI implementation
- LiteLLM integration for unified provider abstraction
- Portkey AI integration for observability
- Comprehensive CLI management interface with interactive settings menu
- Rate limit header tracking with state management
- Benchmark-based model selection
- Academic task classification
- Cross-platform setup scripts (Linux/Mac and Windows)
- Database migrations with Alembic
- Comprehensive test suite

### Changed
- Migrated from TypeScript/Node.js to Python/FastAPI
- Rebranded from "Anna's Gift" to "Lumni"
- Removed VPN rotation features
- Updated all documentation for Python implementation

### Removed
- TypeScript implementation and build artifacts
- VPN rotation infrastructure
- Old development documentation and artifacts
- Legacy test files

### Fixed
- Rate limit header tracking implementation
- Cross-platform path handling
- Database schema consistency

## [1.0.0] - 2024-12-05

### Added
- Initial TypeScript implementation
- Basic provider abstraction
- Fallback system
- VPN rotation
- Usage tracking
